# Histoire Svelte

This is a [histoire.dev](https://histoire.dev) demo book using [svelte.dev](https://svelte.dev)

## Install

```
degit oneezy/histoire-svelte
pnpm i
```

## Development
```
pnpm story:dev
```

## Production
```
pnpm story:build
pnpm story:preview
```

## Cleanup
To remove `node_modules` and `build` files
```
pnpm clean
```
